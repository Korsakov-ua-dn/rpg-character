## Требования

- Node.js >= 16

## Установка

`npm install`

## Запуск для разработки

В режиме разработки (development) приложение запускается командой:

`npm start`

## Сборка для продакшена

Для публикации проекта на сервере сначала выполняется сборка приложения командой:

`npm run build`
